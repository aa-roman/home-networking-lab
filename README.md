# home-networking-lab

## Overview
This repository documents my hands-on IT support and networking lab environment built using VirtualBox, Windows, Ubuntu, Wireshark, and Cisco Packet Tracer.

The goal of this lab is to practice:
- Virtual machine administration
- Networking troubleshooting
- DNS/DHCP/NAT diagnostics
- Firewall configuration
- Packet analysis
- Routing and subnetting
- Professional troubleshooting documentation

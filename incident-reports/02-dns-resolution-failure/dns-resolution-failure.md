# DNS Resolution Failure

## Incident
User reported websites were inaccessible by name.

## Symptoms
- ping google.com failed
- ping 8.8.8.8 succeeded

## Troubleshooting Process
1. Verified internet connectivity
2. Tested external IP communication
3. Identified DNS name resolution failure
4. Inspected IPv4 DNS configuration

## Root Cause
Invalid DNS server address configured manually.

## Resolution
Restored automatic DNS configuration.

## Skills Practiced
- DNS troubleshooting
- IPv4 configuration
- Connectivity diagnostics

## Incident
Device experienced connectivity issues after manual network configuration.

## Symptoms
- Internet connectivity failure
- External hosts unreachable

## Troubleshooting Process
1. Verified local IP configuration
2. Inspected subnet mask settings
3. Tested default gateway reachability
4. Verified DNS settings

## Root Cause
Incorrect subnet mask and gateway configuration.

## Resolution
Restored proper subnet mask and default gateway values.

## Skills Practiced
- Static IP configuration
- Gateway troubleshooting
- Subnet troubleshooting
- Connectivity diagnostics

# DNS Packet Analysis

## Incident
User reported inability to access websites.

## Symptoms
- Websites failing to load
- DNS resolution failure suspected

## Troubleshooting Process
1. Captured live network traffic using Wireshark
2. Filtered DNS traffic
3. Inspected DNS query packets
4. Verified missing DNS responses

## Root Cause
DNS resolution failure preventing domain lookups.

## Resolution
Restored valid DNS configuration.

## Skills Practiced
- Packet capture analysis
- DNS troubleshooting
- Traffic filtering
- Protocol inspection

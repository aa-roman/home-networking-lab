# Subnetting and Gateways

## Subnet Mask
Defines which devices belong to the local network.

## Default Gateway
Allows traffic to leave the local network and reach external destinations.

## Common Symptoms

### Incorrect Gateway
- No internet access
- External IPs unreachable

### Incorrect Subnet Mask
- Communication issues
- Routing problems
- Connectivity instability

## Troubleshooting Flow
1. Verify IP address
2. Verify subnet mask
3. Verify gateway
4. Test gateway reachability
5. Test external connectivity

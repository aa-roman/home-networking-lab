# Advanced Packet Analysis

## ARP
ARP maps IP addresses to MAC addresses on local networks.

## DHCP DORA Process
1. Discover
2. Offer
3. Request
4. ACK

## ICMP
Ping uses ICMP Echo Requests and Echo Replies.

## TCP Handshake
TCP establishes communication using:
- SYN
- SYN/ACK
- ACK

## Retransmissions
Repeated retransmissions may indicate:
- connectivity instability
- unreachable hosts
- firewall issues

## Troubleshooting Workflow
1. Capture traffic
2. Filter protocols
3. Inspect requests/responses
4. Identify missing communication
5. Determine likely failure point

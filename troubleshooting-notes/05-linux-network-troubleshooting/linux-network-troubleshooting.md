# Linux Network Troubleshooting

## Commands

### ip addr
Displays interfaces and IP addresses.

### ping
Tests connectivity.

### traceroute
Displays routing path.

### ss
Displays active network sockets and connections.

## Common Troubleshooting Workflow
1. Verify interface status
2. Verify IP assignment
3. Test external IP connectivity
4. Verify DNS resolution
5. Inspect routing behavior

## DNS Troubleshooting
If IP connectivity works but domain names fail:
- DNS configuration may be broken.

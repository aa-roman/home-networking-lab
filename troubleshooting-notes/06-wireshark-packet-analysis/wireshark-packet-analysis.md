# Wireshark Packet Analysis

## DNS Traffic
DNS packets contain:
- queries
- responses
- resolved IP addresses

## TCP
TCP provides reliable, connection-oriented communication.

## UDP
UDP provides fast, connectionless communication.

## Common Ports
- 53 = DNS
- 80 = HTTP
- 443 = HTTPS
- 22 = SSH
- 3389 = RDP

## Troubleshooting Workflow
1. Capture traffic
2. Apply filters
3. Identify protocols
4. Inspect packet behavior
5. Identify failures or abnormal traffic

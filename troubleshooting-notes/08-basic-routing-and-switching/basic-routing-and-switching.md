# Basic Routing and Switching

## Switches
Switches connect devices within the same network using MAC addresses.

## Routers
Routers connect different networks using IP addresses.

## Same-Subnet Communication
Devices in the same subnet can communicate directly through switches.

## Inter-Network Communication
Communication between different networks requires routing.

## Default Gateway
The default gateway allows devices to communicate outside their local subnet.

## Common Troubleshooting Scenarios
- Incorrect subnet masks
- Wrong IP assignments
- Incorrect gateways
- Disabled router interfaces
- Missing routing configuration
